⚡ Understanding the Comparison

Move the slider to see how the functions behave at different values of x. Notice:

1. Polynomial growth: The change itself is a polynomial of one degree less. For x², the change is linear (2x + 1). For x³, the change would be quadratic. You'll see this pattern clearly in the Finite Differences table in the Explorer tab.

2. Exponential growth: The change always equals a constant times the current value. For 2ˣ, each change equals the current value (multiplier = 1). For 3ˣ, each change equals 2× the current value.

Two Ways to Analyze Growth

🔢 Using Differences

Look at the Finite Differences table above. For polynomials, repeated differences eventually become constant (x² reaches constant at Δ²f, x³ at Δ³f). For exponentials, each difference equals (base - 1) times the current value. For 2ˣ, that's (2-1) = 1 times the value, so differences equal the values themselves. For 3ˣ, that's (3-1) = 2 times the value.

📊 Using Ratios

Look at the Ratio columns in the comparison table. For 2ˣ, f(x+1)/f(x) always equals 2.00. For x², the ratio decreases: 2.25, 1.78, 1.56... approaching 1 as x gets large.

🎯 Tips for Graph Recognition

Polynomials (x², x³):

  • x²: U-shaped parabola, symmetric around x = 0
  • x³: S-shaped curve, passes through origin, different on each side

Exponentials (2ˣ, 3ˣ, eˣ):

  • Growth: Always positive, increases rapidly for positive x
  • Decay ((1/2)ˣ): Always positive, decreases toward 0 for positive x
  • All pass through point (0, 1) since any number to power 0 equals 1
