üìê Essential Theory

Taylor Expansion with Remainder

If $f$ is sufficiently smooth near $x = a$, then for small $h$:

$$f(a+h) = f(a) + hf'(a) + \frac{h^2}{2}f''(a) + \cdots + \frac{h^n}{n!}f^{(n)}(a) + R_n(h)$$

where the remainder term satisfies $R_n(h) = O(h^{n+1})$.

Truncation Error

When a numerical method replaces an infinite expansion by finitely many terms, the neglected remainder is called the truncation error. Its order describes how fast the error decreases as the step size decreases.

Order Revisited

If a numerical approximation $A(h)$ satisfies

$$A(h) = A + Ch^p + O(h^{p+1})$$

then the method is said to be of order $p$.

Key Idea

Order of convergence is not magic: it comes directly from Taylor expansions and the size of the neglected terms.

Richardson Extrapolation

Given two approximations with step sizes $h$ and $h/2$:

$$A(h) = A + Ch^p + O(h^{p+1})$$ $$A(h/2) = A + C\left(\frac{h}{2}\right)^p + O(h^{p+1})$$

We can eliminate the leading error term and form a more accurate approximation:

$$A_{\text{rich}} = \frac{2^p A(h/2) - A(h)}{2^p - 1}$$
Key Idea

Extrapolation is a deliberate "error-cancellation move". You run the same method twice at two step sizes, then combine the answers.

Problem 5: Repeated Extrapolation (The Romberg Idea)

Suppose a numerical method produces approximations

$A(h) = A + Ch^2 + O(h^4)$
$A(h/2) = A + C(h/2)^2 + O(h^4)$
$A(h/4) = A + C(h/4)^2 + O(h^4)$
  1. Use Richardson extrapolation to form $R_{1,1} = \frac{4A(h/2) - A(h)}{3}$.
  2. Form $R_{2,1} = \frac{4A(h/4) - A(h/2)}{3}$.
  3. Extrapolate again to obtain $R_{2,2} = \frac{16R_{2,1} - R_{1,1}}{15}$.
  4. What is the order of the leading error term in $R_{2,2}$?
