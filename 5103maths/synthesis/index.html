üî¨ One Universal Diagnostic: Step-Halving

If a method has global order $p$, halving $h$ reduces the dominant error by about $2^p$.

$$\text{Error}(h/2) \approx \frac{\text{Error}(h)}{2^p}$$
Order $p$ Error Reduction Examples
1 Halves (√∑2) Euler, Bisection
2 Quarters (√∑4) Trapezium, Midpoint, Modified Euler, Newton*
4 16√ó smaller (√∑16) Simpson, RK4
6 64√ó smaller (√∑64) Romberg (second extrapolation)

*Newton has quadratic convergence in error ratio, not step size.

Whether you're finding roots, computing integrals, or solving ODEs, the underlying machinery is always the same: understand the error structure via Taylor, then exploit it.

üîó Key Connections to Remember

  • Simpson's rule = Trapezium rule + Richardson extrapolation (order 2 ‚Üí 4)
  • Modified Euler = Euler + averaging slopes (order 1 ‚Üí 2)
  • Romberg = Repeated Richardson extrapolation on trapezium
  • RK4 = Weighted average of four slope samples (order 4)
  • ODE solvers are quadrature rules: $y(t) = y(t_0) + \int_{t_0}^{t} f(s, y(s))\,ds$
