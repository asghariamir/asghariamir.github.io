üìê Essential Theory

Numerical Integration Problem

Given a continuous function $f$ on $[a, b]$, we seek an approximation to

$$\int_a^b f(x)\, dx$$

Midpoint Rule

Partition $[a, b]$ into subintervals of width $h = (b-a)/n$. The midpoint rule is

$$M_n = h \sum_{k=1}^{n} f\left(a + \left(k - \frac{1}{2}\right)h\right)$$

Trapezium Rule

The trapezium rule is

$$T_n = \frac{h}{2}\left[f(a) + 2\sum_{k=1}^{n-1} f(a + kh) + f(b)\right]$$

Error via Taylor Expansion

Using Taylor expansions, one finds that both the midpoint and trapezium rules have global error of order 2:

$$\int_a^b f(x)\,dx - M_n = O(h^2), \quad \int_a^b f(x)\,dx - T_n = O(h^2)$$

Simpson's Rule (via Extrapolation)

Take two trapezium approximations with step sizes $h$ and $h/2$:

$$T(h) = I + Ch^2 + O(h^4), \quad T(h/2) = I + C\left(\frac{h}{2}\right)^2 + O(h^4)$$

Applying Richardson extrapolation with $p = 2$ gives

$$S(h) = \frac{4T(h/2) - T(h)}{3}$$

This approximation is called Simpson's rule and has error of order 4.

Key Idea

Simpson is not "a new rule". It is the trapezium rule + one extrapolation step.

Romberg Integration

Romberg integration organises repeated Richardson extrapolation applied to the trapezium rule in a triangular table, producing successively higher-order approximations.
