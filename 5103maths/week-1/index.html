üìê Essential Theory

Exact vs Numerical Solutions

In many mathematical problems, an exact solution either does not exist in closed form or is not practical to compute. In such cases, we seek a numerical approximation: a number that is close enough to the true solution for a given purpose.

Error

If $x^*$ denotes the true value and $x$ an approximation, then:

  • Absolute error: $|x - x^*|$
  • Relative error: $\displaystyle\frac{|x - x^*|}{|x^*|}$ (when $x^* \neq 0$)

Root-Finding Problem

Given a function $f(x)$, the root-finding problem consists of finding $x$ such that $f(x) = 0$.

Bisection Method

If a function $f$ is continuous on an interval $[a, b]$ and $f(a)$ and $f(b)$ have opposite signs, then there exists at least one root in $(a, b)$. The bisection method repeatedly halves the interval, keeping the subinterval in which the sign change occurs.

$$\text{Error bound after } n \text{ steps: } \frac{b-a}{2^n}$$
Key Idea

Numerical methods do not aim to find the answer, but an answer whose error we understand and can control.
