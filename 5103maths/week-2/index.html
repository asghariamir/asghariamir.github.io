üìê Essential Theory

Fixed Points

A number $x^*$ is called a fixed point of a function $g$ if

$$x^* = g(x^*)$$

From Roots to Fixed Points

A root of $f(x) = 0$ can often be found by rewriting the equation in the form $x = g(x)$. Any solution of this equation is a fixed point of $g$.

Example: $x^3 - x - 1 = 0$ can be rewritten as:

  • $x = \sqrt[3]{x + 1}$ (take cube root)
  • $x = x^3 - 1$ (rearrange differently)

Fixed-Point Iteration

Starting from an initial guess $x_0$, define a sequence:

$$x_{n+1} = g(x_n), \quad n = 0, 1, 2, \ldots$$

If the sequence converges to $x^*$, then $x^*$ is a fixed point of $g$.

Convergence and Divergence

An iteration converges if $x_n \to x^*$ as $n \to \infty$. It diverges if the sequence does not approach a finite limit.

Linear Convergence (Informal)

If the error satisfies

$$|x_{n+1} - x^*| \approx C|x_n - x^*|$$

for some constant $0 < C < 1$, the convergence is called linear.

Key Idea

A numerical method is an iterative process. Understanding its behaviour matters more than writing down its formula.

Problem 3: Alternative Rearrangements (Core 2)

The equation $x^3 - x - 1 = 0$ can be rearranged in many ways. Consider:

$$g_1(x) = \sqrt[3]{x + 1}, \qquad g_2(x) = x^3 - 1$$
  1. Use the same starting value $x_0 = 1$.
  2. Perform three iterations for each $g$.
  3. Compare the behaviour of the two sequences.
Key Observation

The same equation can give radically different iteration behaviour depending on how you rearrange it. The choice of $g(x)$ matters!
