üìê Essential Theory

Initial Value Problem (IVP)

A first-order IVP has the form

$$y'(t) = f(t, y), \quad y(t_0) = y_0$$

We seek approximations to $y(t)$ at discrete time points.

Euler's Method

Using the tangent at $(t_n, y_n)$:

$$y_{n+1} = y_n + h \cdot f(t_n, y_n)$$

Euler's method has global error of order 1.

Modified Euler (Heun's) Method

Predict using Euler, then correct using an average slope:

$y^* = y_n + h \cdot f(t_n, y_n)$  (predictor)

$y_{n+1} = y_n + \frac{h}{2}\left[f(t_n, y_n) + f(t_{n+1}, y^*)\right]$  (corrector)

This method has global error of order 2.

Runge‚ÄìKutta Method of Order 4 (RK4)

Define four slope estimates:

$k_1 = f(t_n, y_n)$

$k_2 = f\left(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_1\right)$

$k_3 = f\left(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_2\right)$

$k_4 = f(t_n + h, y_n + h k_3)$

Then:

$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$

RK4 has global error of order 4.

Key Idea

ODE solvers are quadrature rules applied to the differential equation $y'(t) = f(t, y)$. This connects everything back to Week 6!
