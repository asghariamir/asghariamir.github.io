The Key Insight

üí° Why Newton Wins

Bisection always has C = 0.5 because each step halves the interval (error bound). Newton achieves quadratic convergence by using the tangent line, which makes the error square itself each step.

The derivative |g'(x*)| isn't just about whether fixed-point converges ‚Äî it tells you exactly how many digits you gain per step: ‚àílog‚ÇÅ‚ÇÄ(|g'(x*)|).

*Note: The fixed-point iteration g(x) = ¬Ω(x + 2/x) is actually Newton's method in disguise! It achieves quadratic convergence because g'(x*) = 0 at the fixed point.
