2. Stable Fixed Point vs Period Doubling

Compare a = 2.8 (stable) with a = 3.25 (2-cycle), both with b = 1.

At a = 2.8: f'(x*) = 2 − 2.8 = −0.8. Since |−0.8| < 1, the fixed point is stable. At a = 3.25: f'(x*) = −1.25, so |−1.25| > 1 — unstable. Switch to cobweb view to see the difference.
The bifurcation happens at a = 3 exactly, where |f'(x*)| = 1. Below 3: spiral inward. Above 3: the fixed point repels, and a stable 2-cycle catches the orbit. The Bifurcation tab shows this as the first fork.

3. Sensitive Dependence

Set a = 3.8, b = 1. Add two orbits: x₀ = 0.1 and x₀ = 0.1001. After how many steps do they diverge?

In the chaotic regime, all periodic orbits are unstable. Nearby starting points separate exponentially — this is sensitive dependence on initial conditions, the hallmark of chaos. The bifurcation diagram shows periodic windows inside the chaos: islands of order in a sea of complexity.

4. The Role of b

Compare a = 1.5, b = 0.5 with a = 1.5, b = 1. Both converge — but to different values. Why?

The fixed point is x* = (a−1)/(ab). With a = 1.5, b = 0.5: x* = 0.5/0.75 = 2/3. With a = 1.5, b = 1: x* = 0.5/1.5 = 1/3. The parameter b rescales the carrying capacity — it changes where the fixed point sits, but the stability (f'(x*) = 2 − a = 0.5) is the same in both cases.
