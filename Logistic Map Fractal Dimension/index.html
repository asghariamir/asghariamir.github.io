Logistic Map Explorer

Purpose: This interactive tool visualizes the behavior of the logistic map and estimates its fractal dimension using the ε-box counting method. The logistic map is a classic example in chaos theory and mathematical biology.

How to use:

  1. Use the sliders to set the parameters: r (growth rate), x₀ (initial condition), k (controls ε = 2⁻ᵏ).
  2. Visualize the attractor and ε-box coverage below.
  3. Add ε-table rows to estimate N(ε) and fit a line in log-log space to approximate the fractal dimension.
