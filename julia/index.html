Julia Sets & the Mandelbrot Set

When we iterate a simple rule on complex numbers, the boundary between orbits that stay bounded and those that escape to infinity forms breathtaking fractal structures.

The Iteration Rule

Pick a starting point in the complex plane. Apply the rule repeatedly. The parameter is a fixed complex number that shapes the Julia set.

Escape Criterion
