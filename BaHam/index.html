      ویرایش صفحه

      متن‌ها را تغییر دهید، سپس «ذخیرهٔ صفحه» را بزنید تا فایل تازه دانلود شود. برای پررنگ از **متن** استفاده کنید.

      چگونه منتشر کنیم؟ / How to publish

      فارسی

      1. دکمهٔ «ذخیرهٔ صفحه» بالا را بزنید؛ فایل index.html تازه دانلود می‌شود.
      2. به مخزن گیت‌هابِ سایت بروید (همان‌جا که index.html فعلی هست).
      3. روی Add file ← Upload files بزنید و فایل تازه را بکشید و رها کنید (نامش باید همان index.html بماند تا جایگزین شود).
      4. پایین صفحه، Commit changes را بزنید.
      5. کمتر از یک دقیقه بعد، صفحه به‌روز می‌شود.

      English

      1. Click “Save page” above; a fresh index.html downloads.
      2. Go to the site’s GitHub repository (where the current index.html lives).
      3. Click Add file → Upload files and drag the new file in (keep the name index.html so it replaces the old one).
      4. At the bottom, click Commit changes.
      5. In under a minute, the page updates.

      فقط کسانی که به مخزن گیت‌هاب دسترسی دارند می‌توانند منتشر کنند؛ دیگران فایلِ دانلودشده را برای کسی که دسترسی دارد می‌فرستند. رمز ویرایش فقط جلوی ویرایشِ اتفاقی را می‌گیرد، نه امنیتِ واقعی. — Only people with GitHub repo access can publish; others can send the downloaded file to someone who has access. The passphrase only prevents accidental edits, it is not real security.
